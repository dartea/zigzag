ZigZag

Usage :

Include script after the jQuery library : 
<script src="/path/to/jquery.zigzag.min.js"></script>
then in your code do: 

     $(".item").ZigZag({
        direction      : 'top', //can be : top,right,bottom,left
        item_target    : '#mytargetdiv',//(must be unique)
        onFinish       : myCallBack //a callback function  calling at the end of animation
     });
You can set a callback function : 
 function myCallBack(item){
        //do something at the end of animation
        console.log(item);
        console.log(item.data());
  }
